# Whiteboard

## Quick Start (Solo — No Backend Needed)

Just open `whiteboard.html` directly in your browser. No server, no npm, no install.

```
Double-click whiteboard.html
```

That's it. Everything works: draw, undo/redo, zoom, clip & share.

---

## With Collaboration (Backend Required)

```bash
cd be
npm install
npm start
```

Then open: http://localhost:3001

Click **Collaborate**, enter your name, and share the Room ID.

---

## Features

- ✏️ Pencil, Line, Arrow, Rectangle, Diamond, Circle, Text, Eraser, Laser
- 🎨 Stroke & fill colors, stroke width, roughness, opacity
- 🔤 Font family & size selector with live preview
- ↩️ Undo / Redo (Ctrl+Z / Ctrl+Y)
- 🔍 Zoom (scroll) + Pan (Alt+drag)
- ✂️ Clip & Share — drag to select any region, copy or download as PNG
- 👥 Real-time collaboration with live cursors (needs backend)
- ⌨️ Keyboard shortcuts: V R D C A L P T E Z
